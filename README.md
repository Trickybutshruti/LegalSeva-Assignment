# LegalSeva-Assignment

Overview
This project implements two AI agents using Python and Large Language Models.

--------------------------------------------------

Task 1 – CEO Research Agent

This agent autonomously researches a CEO using internet sources.

Workflow:
1. Searches the internet for information
2. Reads relevant webpages
3. Extracts useful content
4. Uses an LLM to generate a structured report

Output includes:
- Background
- Career journey
- Leadership style
- Major achievements

Technologies Used:
Python
DuckDuckGo Search (ddgs)
BeautifulSoup
Requests
Groq LLM API

Run command:

python agent_main.py

--------------------------------------------------

Task 2 – AI Code Analysis Agent

This agent analyzes Python code and provides intelligent feedback.

Capabilities:
- Reads a Python file
- Explains what the code does
- Identifies potential issues
- Suggests improvements

Run command:

python code_agent.py
